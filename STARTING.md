För att kunna köra det här projektet behöver man synkronisera med Expo Go.
Hur fixar man setup? Länk: https://www.notjust.dev/blog/2021-11-04-expo-development-environment-setup-for-first-react-native-project-windows

Vi har redan ett projekt så efter du sattit upp installationen på datorn, så kan du skriva:

"npm install --global expo-cli"

Sedan har jag upplevt att hotspot funkar bäst för mig därefter kör jag:

npx expo start --tunnel (Men det funkar även med "npx expo start").
